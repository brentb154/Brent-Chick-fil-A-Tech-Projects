Payroll Management System

Complete payroll solution with overtime tracking, PTO management, uniform ordering, and year-end processing.

Google Apps Script Google Sheets Web App
This is a Google Apps Script application. It runs inside Google Sheets and cannot be opened directly in a browser. Follow the setup steps below to deploy it to your Google account.

Features

  • Multi-location overtime tracking with alerts and trend analysis
  • CSV timecard upload and pay period processing
  • Employee-level OT history and lookups
  • Uniform ordering with payroll deduction scheduling
  • PTO request portal and tracking
  • Payroll calendar and bi-weekly processing
  • Year-end processing wizard
  • System health monitoring and reporting

Setup Steps

  1. Create a new Google Sheet and name it CFA Payroll System.
  2. Open Extensions → Apps Script in the sheet.
  3. Copy the code from each .gs file (Code.gs, DashboardModule.gs, PTOModule.gs, ReportsModule.gs) into matching script files.
  4. Copy each .html file into matching HTML files in the Apps Script editor.
  5. Run initializeSystem once to set up the sheet tabs and structure.
  6. Deploy: Deploy → New Deployment → Web App. Set access to your organization and copy the URL.
  7. See COMPLETE_SETUP_GUIDE.md for full step-by-step instructions.
