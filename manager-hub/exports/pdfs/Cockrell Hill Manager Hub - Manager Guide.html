CFA Accountability System - Manager Guide

Version: 2026-01-24
Audience: Managers
Purpose: Day-to-day usage of the accountability system

Getting Started

Login

  • Navigate to the web app URL provided by your director.
  • Select your role: Manager.
  • Enter the manager password.
  • Click Login.

If you see a session error, refresh the page and try again. If it persists, contact a director.

Password Reset Process

  • Managers cannot reset their own password.
  • Contact a director to reset the shared manager password.
  • After reset, log out and back in with the new password.

Dashboard Overview

The Manager Dashboard includes: - Quick Stats: daily infractions, employees at risk, your entries this month. - Main Menu: Add Infraction, View Employees, My Activity. - Quick Links: company resources (if configured).

Mobile Usage

  • The system is mobile-friendly via browser.
  • Use the main dashboard menu for quick navigation.
  • If the screen looks condensed, rotate to landscape or use the menu buttons.

Adding Infractions (Step-by-Step)

1) Open the Infraction Form

From the dashboard, select Add Infraction.

2) Select the Employee

  • Use the search input or list.
  • Confirm the employee name and ID.

3) Choose Infraction Type

Select the appropriate bucket from the list. If unsure, ask a director for guidance.

4) Write the Description (Best Practices)

Descriptions must be at least 240 characters. Include: - What happened - When it happened - Who was involved - Any coaching provided

Example format:

“At 3:15 PM during the lunch rush, the employee failed to follow handoff procedures for four drive-thru orders. I observed the behavior directly, coached the employee immediately, and reviewed the correct handoff steps. The employee acknowledged the error and agreed to slow down and verify each order before passing to expo.”

5) Backdating Infractions

Backdating is limited to the last 7 days. If you need to record an older issue, notify a director.

6) Submit and Confirm

Click Submit. The system will show confirmation or validation errors.

Viewing Employee Records

Searching for Employees

Use the Employee List page: - Search by name or ID. - Filter by location or status if available.

Understanding Point Totals

  • Point totals reflect active infractions only.
  • Points expire after 90 days.
  • Positive behavior credits reduce point totals.

Reading Infraction History

Employee detail page shows: - Date of infraction - Type - Points assigned - Status (active/expired)

Color Coding

If the UI uses color indicators: - Green: normal / low risk - Yellow: approaching threshold - Red: high risk / urgent

Understanding Consequences

Thresholds

Common threshold levels: - 6 points: coaching / warning - 9 points: final warning / probation - 15 points: termination review

Exact policies are defined by leadership. Ask a director if unsure.

Manager Responsibilities

  • Document promptly and accurately.
  • Coach employees in real time.
  • Escalate to director when thresholds are crossed.

Best Practices

  • Document the same day when possible.
  • Use consistent wording and specifics.
  • Avoid emotional or subjective language.
  • Double-check employee selection and location.

Common Mistakes to Avoid

  • Short descriptions (less than 240 characters)
  • Backdating beyond 7 days
  • Selecting the wrong employee
  • Missing the location field

Getting Help

Who to Contact

Primary: Director on duty
Secondary: Operations leader or system admin

Reporting Issues

Report: - Page errors or timeouts - Incorrect point totals - Missing employees

Include: - Screenshot - Time of issue - What you were doing

FAQ

Q: I can’t find an employee.
A: They may be inactive in Payroll Tracker. Ask a director to verify status.

Q: My infraction didn’t submit.
A: Check for validation errors and session expiration. Refresh and retry.

Q: Why are points lower than expected?
A: Infractions may have expired or positive credits reduced totals.
