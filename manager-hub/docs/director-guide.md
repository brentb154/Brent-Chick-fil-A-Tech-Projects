# CFA Accountability System - Director Guide

Version: 2026-01-24  
Audience: Directors / Operators  
Purpose: Administration, configuration, and oversight

---

## Includes Everything in Manager Guide
Directors should be familiar with all Manager workflows:
- Login and dashboard
- Adding infractions
- Viewing employees and histories
- Thresholds and consequences
- Best practices

---

## User Management

### Adding Managers
- Open **User Management** page.
- Add user name, email, and role.
- Confirm permissions and save.

### Changing Roles
- Use the user list to update role (Manager/Director/Operator).
- Save and notify the user.

### Deactivating Users
Deactivate managers who are no longer active.
This prevents access but preserves audit history.

---

## Settings Configuration

### Email Settings
Settings page controls:
- Director notification emails
- Termination alerts
- System notification preferences

### Changing Passwords
Update shared passwords in the Settings sheet:
- Manager password
- Director password
- Operator password

### Threshold Modifications
Thresholds are typically 6/9/15 points.
If policy changes:
- Update Settings thresholds
- Communicate changes to managers

### Bucket Customization
Infraction buckets can be adjusted in Settings.
Keep definitions consistent with training.

---

## Director Actions

### Editing Infractions
Use **Edit Infraction** to correct:
- Employee
- Date
- Type
- Description

### Removing Points
Use the **Remove Points** workflow:
- Requires reason and director name
- Creates an audit entry

### Positive Behavior Credits
Add positive credits to reward improved performance.
Credits appear in history and reduce totals.

### Probation Extensions
Use probation tools to extend or update probation as needed.

---

## Reports and Analytics

### Generating Reports
From **Reports**:
- Choose report type
- Select date range
- Select format (PDF/Excel/CSV)

### Scheduling Recurring Reports
Create scheduled reports for:
- Weekly leadership updates
- Monthly compliance reviews

### Interpreting Data
Use analytics dashboard:
- Threshold distribution
- Infraction trends
- Manager accountability

---

## Employee Termination

### Termination Workflow
When points reach termination threshold:
- Document thoroughly
- Notify leadership
- Export relevant report(s)

### Required Documentation
Ensure:
- Infraction history is complete
- Descriptions meet requirements
- Manager coaching notes included

### Archive Management
Terminated employees should be archived.
Maintain records for compliance.

---

## System Maintenance

### Backups
- Run manual backup when needed.
- Confirm the backup log entry.

### Monitoring System Health
Use **System Status** page to:
- Run health checks
- Review active issues
- Monitor error rates and triggers

### Resolving Issues
Click **Resolve** on an issue to:
- Enter resolution notes
- Mark the event as resolved
- Keep audit trail consistent

### Escalation
Escalate to system admin when:
- Data integrity errors are found
- External dependencies fail
- Email sending is blocked by permissions

---

## FAQ

**Q: Why do I see “trigger missing”?**  
A: The trigger wasn’t created or was deleted. Re-run the scheduling function.

**Q: Email quota error appears on System Status.**  
A: Ensure the script is authorized to send email and quota is available.

**Q: A manager can’t see an employee.**  
A: Verify employee is active in Payroll Tracker and permissions are correct.
