# Mobile Quick Start

## Login
- Open the web app link in your mobile browser.
- Select your role and enter password.

## Add Infraction
- Tap **Add Infraction** from dashboard.
- Fill required fields and submit.

## View Employees
- Tap **View Employees**.
- Use search to find employee records.

## Tips
- Rotate to landscape for larger tables.
- If screen is slow, close other tabs.
