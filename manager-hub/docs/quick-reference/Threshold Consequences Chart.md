# Threshold Consequences Chart

| Points | Level | Typical Action |
|---|---|---|
| 0-5 | Normal | Standard coaching |
| 6 | Warning | Manager coaching + director notice |
| 7-8 | At Risk | Monitor closely |
| 9 | Final Warning | Probation + director review |
| 10-14 | High Risk | Escalate and monitor |
| 15+ | Termination Review | Formal review + documentation |

*Exact thresholds may be adjusted by leadership.*
