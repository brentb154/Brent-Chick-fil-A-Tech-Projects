# Director Actions Cheat Sheet

## User Management
- Add/update roles
- Deactivate users

## Infraction Administration
- Edit infractions
- Remove points (requires reason)
- Add positive credits
- Extend probation

## Reports & Analytics
- Generate reports (PDF/Excel/CSV)
- Schedule recurring reports
- Review trends and accountability

## System Health
- Open **System Status**
- Run health checks
- Review and resolve issues
