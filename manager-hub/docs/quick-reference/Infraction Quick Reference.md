# Infraction Quick Reference

## Point Values (Typical)
- Minor: 1 point
- Moderate: 3 points
- Major: 5 points
- Severe: 8 points

## Required Fields
- Employee
- Date
- Infraction Type
- Points Assigned
- Description (min 240 chars)
- Location
- Entered By

## Description Checklist
- What happened
- When and where
- Who was involved
- Coaching provided

## Backdating Rule
Allowed up to 7 days in the past.
