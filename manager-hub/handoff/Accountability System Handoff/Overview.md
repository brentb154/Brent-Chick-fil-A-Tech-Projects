# Accountability System Handoff - Overview

## Project Background
This system provides a structured, auditable way to record employee accountability events, calculate points, and notify leadership.

## Problem It Solves
- Consistent documentation
- Clear threshold escalation
- Centralized reporting
- Audit trail of actions

## High-Level Architecture
- Google Apps Script backend
- Google Sheets datastore
- Web app UI served from Apps Script
- Email notifications via Gmail
- Scheduled triggers for automation
