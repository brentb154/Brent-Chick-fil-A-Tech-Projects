Manager Accountability Hub In Development

Employee management system for tracking performance, infractions, and team accountability across locations.

Google Apps Script Google Sheets Web App
This is a Google Apps Script application. It runs inside Google Sheets and cannot be opened directly in a browser. Follow the setup steps below to deploy it to your Google account.

Features

  • Employee infraction logging with point-based tracking system
  • Multi-location support (Cockrell Hill DTO, DBU, and more)
  • Manager login and signup with role-based access
  • Employee detail views with full infraction history
  • Employee list with search and filtering
  • User management for manager accounts
  • Director-level dashboard with analytics and risk flags

Setup Steps

  1. Create a new Google Sheet for the accountability system.
  2. Open Extensions → Apps Script in the sheet.
  3. Run Phase1_SheetSetup.gs first — this creates all required sheet tabs and structure.
  4. Copy each .html file into matching HTML files in the Apps Script editor.
  5. Deploy: Deploy → New Deployment → Web App. Set access to your organization.
  6. Share the deployment URL with your management team.
