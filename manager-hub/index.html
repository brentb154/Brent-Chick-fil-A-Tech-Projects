Manager Accountability Hub

Employee management system for tracking performance, infractions, and team accountability. Full web app with login, dashboards, reporting, and system monitoring.

Google Apps Script Google Sheets Web App
This is a Google Apps Script application. It runs inside Google Sheets and cannot be opened directly in a browser. Follow the setup steps below to deploy it to your Google account.

Features

  • Secure login and user management system
  • Employee list with detailed individual profiles
  • Infraction tracking and form submission
  • Advanced reporting and analytics dashboard
  • Probation tracking and management
  • PDF export for documentation
  • System health monitoring and status pages
  • Backup and disaster recovery tools
  • Settings management with change tracking
  • Data quality validation

Setup Steps

  1. Create a new Google Sheet for the Accountability Hub.
  2. Open Extensions → Apps Script and paste each .gs file.
  3. Create all HTML files as listed below.
  4. Run the initial setup function to create required sheet tabs and settings.
  5. Deploy as a Web App (Deploy → New Deployment → Web App).
  6. Configure user accounts and permissions in the Settings tab.
