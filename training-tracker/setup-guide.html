Training Tracker — Setup Guide

From zero to a live training dashboard. It looks like a lot because there are several files to paste, but every step is just copy-paste and clicking buttons. Follow them in order.

⏱ About 30 minutes
This one lives inside a Google Sheet and feeds off a Google Form. Training gets logged through the form, and the Sheet turns it into a live dashboard, timelines, and alerts. You run it from a Training Tools menu that appears right in the Sheet — there's no web app to deploy.
Heads up on the file count: there are 9 script files and 3 small dialog files to paste. That's the only part that feels like a lot — it's repetitive, not hard. Don't let the number scare you off.
1

Create the Google Sheet

  1. Go to sheets.google.com and click Blank spreadsheet.
  2. Name it something clear like "Training Tracker".
  3. Don't add any tabs — Step 3 builds all of them for you.
2

Paste the Files

Click Extensions → Apps Script. Then create each file below and paste in its contents. Match the names exactly — the numbers keep them in order.

Script files (use + → Script):

01_Menu_and_Setup    02_Form_and_Dedup    03_Dashboard
04_Timeline          05_Certification     06_Alerts
07_UI_Functions      08_DataSync          DIAGNOSTIC

HTML files (use + → HTML):

AlertSettingsDialog    DeduplicationSidebar    TimelineDialog

Delete the empty starter myFunction file, then press Ctrl/Cmd+S to save everything.

You do not paste index.html into Apps Script — that's just the web page you're reading this from.
3

Run Initial Setup

  1. Go back to your spreadsheet tab and reload the page (F5). Wait 3–5 seconds.
  2. A new Training Tools menu appears in the menu bar (next to Help).
  3. Click Training Tools → Initial Setup (Run Once).
  4. Google asks to authorize: Continue → pick your account → Advanced → Go to (project) → Allow. (It needs Sheets, Gmail for alerts, and permission to show its dialogs.)

Setup builds every tab — Position Requirements, Daily Training Log, Master Dashboard, Certification Log, Name Deduplication, Alert Settings, and more — and shows a confirmation of what it created.

No Training Tools menu? Give it a few more seconds and reload again. The onOpen function needs a moment to register after you save the files.
4

Create the Google Form

At forms.google.com, make a new form named Daily Training Log with these questions in this exact order:

#QuestionTypeRequired
1DateDateYes
2Trainee NameShort answerYes
3Position TrainedDropdown (recommended)Yes
4HoursShort answer (number)Yes
5On Track?Multiple choice (Yes / No)Yes
6NotesParagraphNo
Order matters. The script reads responses by column position. Keep these six questions in this order. For Position Trained, a dropdown listing your real positions keeps names clean (there's alias handling, but consistent is better).
5

Link the Form to the Sheet

  1. In your form, open the Responses tab → click the green Link to Sheets icon.
  2. Choose Select existing spreadsheet and pick your Training Tracker sheet.
  3. If it offers Daily Training Log, use it. If it makes a new "Form Responses 1" tab, that's fine too.
  4. Submit one test response to confirm it lands in the Sheet.
If the response went into "Form Responses 1", just run Training Tools → Sync Form Data to pull it into the Daily Training Log. The system recognizes the common form-response tab names automatically.
6

Set Up the Triggers

In the Apps Script editor, click the clock icon (Triggers) → + Add Trigger, and add these:

FunctionWhenWhat it does
onFormSubmitFrom spreadsheet → On form submitThe important one. Processes each entry live.
dailyReminderCheckTime-driven → Day → 8–9pmEmails if no training was logged today.
checkInactiveTraineesTime-driven → Day → 6–7amFlags trainees inactive 3+ days.
checkForDuplicatesTime-driven → Day → 7–8amCatches duplicate / misspelled names.
If you only set up one, make it onFormSubmit — that's what keeps the dashboard live. The other three are nice-to-haves you can add anytime.
7

Configure & Verify

  1. Set your positions and required hours in the Position Requirements tab, and the manager emails in Alert Settings (or via Training Tools → Alert Settings).
  2. Run Training Tools → Run Diagnostic — it checks every tab, trigger, and setting and tells you in plain English if anything's off.
✅ A clean diagnostic and a test form response showing up on the dashboard means you're done.

Going Deeper (Optional)

Certification feedback forms, the Monday auto-populate from your schedule, position-name aliases, and daypart customization are all covered in the full SETUP_GUIDE.md. None of it is required to get the core tracker running — add it when you want it.

Troubleshooting

The Training Tools menu never shows up — reload the Sheet and wait a few seconds; onOpen needs a moment after you save. If it still doesn't appear, confirm 01_Menu_and_Setup got pasted and saved.

Form responses aren't reaching the dashboard — run Training Tools → Sync Form Data, and make sure the onFormSubmit trigger exists (Step 6).

Columns look shifted / wrong data on the dashboard — your form questions are out of order. They must be Date, Name, Position, Hours, On Track, Notes (Step 4).

One person shows up as two people — a name got typed two ways. Run Training Tools → Scan for Duplicates (or the dedup sidebar) to merge them.

Not sure what's broken — run Training Tools → Run Diagnostic. It's built to tell you exactly what's missing.
